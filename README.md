# Solitaire Klondike

Juego clásico de solitario con diseño moderno oscuro, desarrollado con opencode + karpathy-coder skill.

## Skills utilizadas
- **karpathy-coder** — revisión de calidad de código
- **behuman** — respuestas auténticas del agente
- **self-improving-agent** — mejora continua del proyecto
- **workflow-builder** — orquestación de tareas

## Cómo jugar
1. Abre `index.html` en tu navegador
2. Arrastra cartas para moverlas entre pilas
3. Construye de A a K en los 4 palos para ganar

## Desarrollado con
- opencode + Ollama (modelo local gratuito)
- HTML/CSS/JS vanilla (sin frameworks)
